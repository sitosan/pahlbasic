# pahlbasic

A Basic Interpreter for ARM processors STM32F405RG, STM32F407VG/ZG and STM32F411CEU.
Allows making of a selfcontained Boot to Basic Computer.

Documentation: read armbasic.pdf (German)

Basic Examples see BasicExamples folder.

NotePad++ Syntax files in NotePadpp\_Syntax Folder



new Releases for STM32F411CEU, STM32F405RG and STM32F407VG/ZG in folder other\_Releases:

Basic411.hex is for STM32F411CEU   (25MHz)

Basic407.hex is for STM32F407VG    (8MHz)

BasicVG.hex  is for STM32F407VG    (8MHz)  alternative Firmware

BasicZG.hex  is for STM32F407ZG    (8MHz)

Basic405.hex is for STM32F405RG	   (8MHz)

BasicPyb.hex is for PyBoardV1.1    (12MHz)
